7This proyect implements an digital signal generator.

9In this practice, we will develop a Digital Signal Generator (DSG) device.
10The DSG device generates 4 different waveforms: sine, triangular, sawtooth, and square.
11The user can select the type of waveform generated by the DSG using a push button.
12Each time the button is pressed, the system switches from one waveform to the next sequentially.
13The amplitude, DC level (offset), and signal frequency are entered using a 4x4 matrix keypad.
14To enter the amplitude, the user will press the A key followed by the desired voltage value in
15millivolts and the D key to finalize. Similarly, to enter the DC level, the user will press
16the B key followed by the desired DC voltage value in millivolts and the D key to finalize.
17Finally, to enter the frequency, the user must press the C key followed by the frequency
18value in Hertz and the D key to finalize. The current values of Amplitude, DC Level, and
19Frequency along with the current waveform will be printed every second via the serial or USB
20interface to a terminal tool.
21
22The Amplitude should be adjustable between 100mV and 2500mV, the DC Level between 50mV and 1250mV.
23The frequency should be adjustable between 1 Hz and 12000000 Hz.
24Therefore, the max range for the value of the signal is: -2450mV to 3750mV.
25
26By default, the DSG device starts generating a sinusoidal signal with an amplitude of 1000 mV,
27DC Level of 500mV, and frequency of 10Hz. The generated signal and its characteristics
28should be able to be verified by a measuring instrument, multimeter, or oscilloscope.
29
30 The DSG device should be implemented using the following hardware components:
31 - RPPS (Raspberry Pi Pico System) board.
32 - 4x4 matrix keypad.
33 - Digital-to-Analog Converter (DAC0808).
34 - Operational Amplifier (LM358).
35 - LED
36
37NOTE: Using Pin.value(), it must take into account that:
38 - If is given a parameter and it is Pin.OUT: it sets the pin to what is given.
39 - If no parameter is given and it is Pin.IN: it returns the current value of the pin.
40 To other cases, its behavior is not defined.

397 def gen_sin(self, t: int):
398 """
399 Generate a sinusoidal signal.
400 """
401 self.valuevalue = int(self.offsetoffset + self.ampamp*sin((2*pi*t)/SAMPLE))
402
403 def gen_tri(self, t: int):
404 """
405 Generate a triangular signal.
406 """
407 if (t <= SAMPLE/2):
408 self.valuevalue = int(self.offsetoffset + (4*self.ampamp*t)/SAMPLE - self.ampamp)
409
410 else:
411 self.valuevalue = int(self.offsetoffset - (4*self.ampamp*t)/SAMPLE + 3*self.ampamp)
412
413 def gen_saw(self, t: int):
414 """
415 Generate a sawtooth signal.
416 """
417 self.valuevalue = int(self.offsetoffset + (2*self.ampamp*t)/SAMPLE - self.ampamp)
418
419 def gen_sqr(self, t: int):
420 """
421 Generate a square signal.
422 """
423 if (t <= SAMPLE/2):
424 self.valuevalue = int(self.offsetoffset + self.ampamp)
425
426 else:
427 self.valuevalue = int(self.offsetoffset - self.ampamp)
428
429 def calculate(self):
430 """
431 Update the value of the signal generator according to the signal state.
432 """
433
434 if not self.enen:
435 return
436
437 if self.ssss == 0:
438 for i in range(SAMPLE):
439 self.gen_sin(i + 1)
440 self.arrayV[i] = self.valuevalue # Offset of the DAC
441 elif self.ssss == 1:
442 for i in range(SAMPLE):
443 self.gen_tri(i + 1)
444 self.arrayV[i] = self.valuevalue # Offset of the DAC
445 elif self.ssss == 2:
446 for i in range(SAMPLE):
447 self.gen_saw(i + 1)
448 self.arrayV[i] = self.valuevalue # Offset of the DAC
449 elif self.ssss == 3:
450 for i in range(SAMPLE):
451 self.gen_sqr(i + 1)
452 self.arrayV[i] = self.valuevalue # Offset of the DAC
